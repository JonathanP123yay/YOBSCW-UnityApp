# WELCOME!!!!

This is the Unity app for **YOBSCW** that allows you to get all of the headset positions and connect it to your Mac and output it in OBS.

The app is designed to run on Meta Quest headsets and stream real-time positional and orientation data to a companion application on macOS, which can then be used inside OBS for virtual cameras, scenes, and effects.

## Links

**OBS**  
https://obsproject.org

**YOBSCW (Main Project)**  
https://github.com/JonathanP123yay/Yeeps-Obs-Webcam

## What this Unity app does
- Reads headset position and rotation
- Reads controller position and rotation (if enabled)
- Sends data to the computer over the network
- Designed to integrate directly with OBS workflows
